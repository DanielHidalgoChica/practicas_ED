Descripción detallada

T.D.A Tablero Una instancia tab del tipo de dato abstracto Tablero es una matriz 9x9 con casillas que identifican las distintas estructuras, es decir, a las filas, columnas y cubo. Esto es de vital importancia, pues tomaremos en muchas ocasiones el nombre estructura para identificar estos tres terminos, ya que en ninguno de ellos se podran repetir casillas con mismo valor.

Lo representaremos como un vector de grupos, que a su vez identifica todas las casillas que tiene el Tablero del Sudoku Killer

Invariante
Todas las casillas deben ser válidas
Num_Casillas = 81
Todos los grupos del vector deben cumplir con su invariante de representación
