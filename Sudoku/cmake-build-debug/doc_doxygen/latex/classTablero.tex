\doxysubsection{Descripción detallada}
T.\+D.\+A \mbox{\hyperlink{classTablero}{Tablero}} Una instancia {\itshape tab} del tipo de dato abstracto {\ttfamily \mbox{\hyperlink{classTablero}{Tablero}}} es una matriz 9x9 con casillas que identifican las distintas estructuras, es decir, a las filas, columnas y cubo. Esto es de vital importancia, pues tomaremos en muchas ocasiones el nombre estructura para identificar estos tres terminos, ya que en ninguno de ellos se podran repetir casillas con mismo {\ttfamily valor}. 

Lo representaremos como un vector de grupos, que a su vez identifica todas las casillas que tiene el \mbox{\hyperlink{classTablero}{Tablero}} del Sudoku Killer \begin{DoxyInvariant}{Invariante}
Todas las casillas deben ser válidas 

Num\+\_\+\+Casillas = 81 

Todos los grupos del vector deben cumplir con su invariante de representación 
\end{DoxyInvariant}
